Le dévlopement

le Javascript

le HtMl sert à créer le contenu du site: elle va contenir uniqument la partie textuelle du site

le PHP

le PHP est un language de programation libre, utlisé pour généré des pages web dynamiques, qui sont donc modifiés en certaines informations(heure,nom de l'utlisateur,formulaire rempli par l'utlisateur etc)."

le Mysql

Mysql est un system de gestion des base données, il permet d'enregister des informations rempli par l'utlisateur (exemple: formulation de creation de compte) pour ensuite etre ré-utlisé au moment appropié (ex:conexion à un site avec indentification) et mot de passe).
